Branches:

Reminder: Once your branch has been merged back to master, delete your old branch. Failing to do so can cause merge conflicts down the line.

Command Line

Make a new branch: git checkout -b [name_of_your_new_branch]

Switch to existing branch: git checkout [name_of_existing_branch]

List all branches: git branch
